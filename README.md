* 很遗憾，由于 Frackin' Universe 更新频繁，两个人并没有足够的精力维护这个项目的频繁更新，现项目停止维护并转交自动构建系统。

#### 自动构建说明

* 自动构建于 Frackin' Universe Release 后自动更新，自动更新最新的 Frackin' Universe 文本，并合并 FAST 分支中已有的文本。

#### 提交拉请求说明

* 任何拉请求请提交至 FAST 分支，其他分支仅用作合并处理。
